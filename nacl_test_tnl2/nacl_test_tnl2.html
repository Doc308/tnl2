test_tnl2 embedded in a web page

You should see a yellow box below, representing the game world. Red rectangles represent buildings in the sample game world and the small blue squares represent players. The demo starts in client mode and will attempt to connect to a local server.

This demo uses the open source Torque Network Library, which provides high performance real-time networking for games and simulations. Please see http://www.libtnl.com/ for more information about the Torque Network Library.
